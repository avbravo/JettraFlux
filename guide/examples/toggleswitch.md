# ToggleSwitch Example

The `ToggleSwitch` widget is part of the JettraEspressoUI framework.

## Usage

```java
import io.jettra.espresso.widgets.ToggleSwitch;
import io.jettra.espresso.widgets.Text;

ToggleSwitch myToggleSwitch = ToggleSwitch.of(
    Text.of("This is a ToggleSwitch widget.")
);
```
