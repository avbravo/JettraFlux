# SessionTimeOutDialog Example

The `SessionTimeOutDialog` widget is part of the JettraEspressoUI framework.

## Usage

```java
import io.jettra.espresso.widgets.SessionTimeOutDialog;
import io.jettra.espresso.widgets.Text;

SessionTimeOutDialog mySessionTimeOutDialog = SessionTimeOutDialog.of(
    Text.of("This is a SessionTimeOutDialog widget.")
);
```
