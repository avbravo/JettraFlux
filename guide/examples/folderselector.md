# FolderSelector Example

The `FolderSelector` widget is part of the JettraEspressoUI framework.

## Usage

```java
import io.jettra.espresso.widgets.FolderSelector;
import io.jettra.espresso.widgets.Text;

FolderSelector myFolderSelector = FolderSelector.of(
    Text.of("This is a FolderSelector widget.")
);
```
