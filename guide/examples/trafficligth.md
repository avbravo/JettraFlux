# TrafficLigth Example

The `TrafficLigth` widget is part of the JettraEspressoUI framework.

## Usage

```java
import io.jettra.espresso.widgets.TrafficLigth;
import io.jettra.espresso.widgets.Text;

TrafficLigth myTrafficLigth = TrafficLigth.of(
    Text.of("This is a TrafficLigth widget.")
);
```
