# CharsDoughnut Example

The `CharsDoughnut` widget is part of the JettraEspressoUI framework.

## Usage

```java
import io.jettra.espresso.widgets.CharsDoughnut;
import io.jettra.espresso.widgets.Text;

CharsDoughnut myCharsDoughnut = CharsDoughnut.of(
    Text.of("This is a CharsDoughnut widget.")
);
```
